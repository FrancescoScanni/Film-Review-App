
# CineScope: Rate your films easily

A responsive web app based on PHP sessions that allows users to rate their own archive of films and submit reviews and ratings for each title. The interface focuses on exploring the film catalog and collecting user opinions to compare perspectives. Everithing is based on a PHP logic managed by storing in files.
